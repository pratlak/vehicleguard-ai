## What is VehicleGuard AI?

VehicleGuard AI is a vehicle insurance premium calculator that uses AI to assess driver risk and generate personalized insurance quotes. Users can get a quote in two ways:

- **Quote Form** — Fill out driver and vehicle details, get instant risk score, premium estimate, and an AI-generated explanation
- **AI Chatbot** — Have a natural conversation with an AI assistant that collects your information and generates a quote

## Architecture

```
User → CloudFront (HTTPS) → S3 (React Frontend)
                          → ALB → ECS Fargate (risk-engine)
                                      → ECS Fargate (rates-service)
                                      → RDS PostgreSQL
                                      → Claude API (Anthropic)
```

### 3 Microservices

| Service | Port | Responsibility |
|---------|------|---------------|
| **risk-engine** | 8080 | Risk scoring, AI explanation, chatbot, quote persistence |
| **rates-service** | 8081 | Base premium rates lookup by vehicle category |
| **frontend** | 3000 | React UI — quote form, results page, chatbot |

## Risk Scoring Algorithm

1. Load 14 configurable risk factors from PostgreSQL
2. Evaluate each factor against driver/vehicle inputs
3. Sum triggered score impacts → normalize to 0-100
4. Lookup base rate from vehicle_base_rates table
5. Apply premium multipliers sequentially
6. Assign risk tier: LOW / MEDIUM / HIGH / VERY_HIGH
7. Call Claude API for AI explanation
8. Save quote to PostgreSQL
9. Return full quote response

### Vehicle Base Rates
| Category | Base Annual Premium |
|----------|-------------------|
| Sedan | $1,100 |
| SUV | $1,300 |
| Truck | $1,400 |
| Electric | $1,600 |
| Sports | $1,800 |
| Luxury | $2,200 |

## API Endpoints

POST /api/v1/chat/message    — Chatbot conversation

## Running Locally

### Prerequisites
- Docker Desktop
- Java 21
- Node.js 20+

### Setup

1. Clone the repository:
```bash
git clone https://github.com/pratlak/vehicleguard-ai.git
cd vehicleguard-ai
```

2. Create `.env` file:
```bash
echo "ANTHROPIC_API_KEY=your-key-here" > .env
```

3. Start all services:
```bash
docker-compose up
```

4. Open browser:
```
http://localhost:3000
```

### Services will start on:
- Frontend: http://localhost:3000
- Risk Engine: http://localhost:8080
- Rates Service: http://localhost:8081
- PostgreSQL: localhost:5432

### Deploy Docker images to ECR
```bash
aws ecr get-login-password --region us-east-2 | docker login --username AWS --password-stdin YOUR_ACCOUNT_ID.dkr.ecr.us-east-2.amazonaws.com

docker buildx build --platform linux/amd64 -t vehicleguard-risk-engine ./risk-engine
docker tag vehicleguard-risk-engine:latest YOUR_ACCOUNT_ID.dkr.ecr.us-east-2.amazonaws.com/vehicleguard-risk-engine:latest
docker push YOUR_ACCOUNT_ID.dkr.ecr.us-east-2.amazonaws.com/vehicleguard-risk-engine:latest
```

### Deploy frontend to S3
```bash
cd frontend && npm run build
aws s3 sync dist s3://vehicleguard-ai-frontend --delete
aws cloudfront create-invalidation --distribution-id E2PSR1U07IG0R4 --paths "/*"
```
